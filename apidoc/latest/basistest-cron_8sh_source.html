13 # @file basistest-cron.sh
14 # @brief Script intended to be run as cron job to perform automated testing.
15 #
16 # This script sets up the environment for the actual master script which
17 # handles the automated testing. Further, it uses qsub to submit testing jobs
18 # to the configured SGE queue, with the SGE options as set in this script.
19 #
20 # Edit this script to change the settings of the automated testing.
21 # The default settings are the ones used for the cron job running on the
22 # cluster of our lab as the 'swtest' user.
